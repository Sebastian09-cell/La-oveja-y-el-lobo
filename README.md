# La Oveja y el Lobo 🍔

Sitio web para restaurante de hamburguesas artesanales con 3 sedes en la Sabana Norte de Bogotá. Ganadores Burger Master 2023-2024.

## 🌐 Demo
https://laovejayellobo-demo.netlify.app/

## ✨ Funcionalidades
- Menú interactivo con filtros por categoría (burgers, pizzas, bebidas)
- Carrito de pedidos con contador y total en tiempo real
- Botones de WhatsApp personalizados por sede
- Horarios de atención por sede
- Diseño responsive para móvil y desktop

## 🛠️ Tecnologías
- HTML5
- CSS3 (variables CSS, Grid, Flexbox)
- JavaScript vanilla
